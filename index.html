HTML

What is HTML?

HTML (Hypertext Markup Language) is the standard language for creating a webpage. It describes the structure of the website. HTML ultilizes an object called tags to add elements to a page, such as text, images, videos, menus, etc.
Here are the tags we will cover today:
  • Paragraphs < p > <\p>
  • Headers < h1 > <\h1> (h1 largest-h6 smallest)
  • Hyperlinks < a href = "url">
  • List < ol > (ordered list) < ul >(unordered list )
  • Images < img src = "" >
  • Div

CSS

What is CSS?

CSS (Cascading Stylesheet) is the language for bringing style, color, formatting, and life to any website. Some of the most common attributes:
  • Color
  • Text Alignment
  • Left, right, top, bottom
  • Width, height
